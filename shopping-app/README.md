# 🛒 ShopEase - Angular E-Commerce Application

ShopEase is a modern e-commerce web application developed using **Angular**, **TypeScript**, **HTML**, **CSS**, and **JSON Server**. It provides a complete online shopping experience with user authentication, product browsing, shopping cart management, checkout, purchase history, and order tracking.

---

## 🚀 Features

- 🔐 User Login & Signup
- 🏠 Home Dashboard
- 📂 Product Categories
- 🛍️ Product Listing
- 🖼️ Product Images
- ➕ Add to Cart
- ➖ Update Cart Quantity
- 🛒 Checkout
- 📦 Order Confirmation
- 📜 Purchase History
- 📄 Order Details
- 👤 User Profile
- 🔔 Popup Notifications
- ⏳ Preloader
- 📱 Responsive Design

---

## 🛠️ Tech Stack

- Angular
- TypeScript
- HTML5
- CSS3
- JSON Server
